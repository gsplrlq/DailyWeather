# pushWeather 云函数

定时推送天气订阅消息的微信云开发云函数。

## 功能

- 每天自动向订阅用户发送天气提醒
- 点击消息可跳转到小程序查看详细天气
- 支持多用户批量推送

## 订阅消息模板

```
{{date1.DATA}} - 日期
{{thing5.DATA}} - 温馨提示（含可爱图标）
```

## 温馨提示文案

| 文案 | 图标 |
|------|------|
| 点击查看今日天气，开启美好一天！ | ☀️ |
| 今日天气已更新，快来看看吧~ | 🌤️ |
| 新的一天，点击查看天气情况！ | 🌈 |
| 天气变化早知道，点击查看~ | 🌧️ |
| 今日天气提醒，点击了解详情！ | ❄️ |

## 配置

### 定时触发器

默认每天 11:00 执行（北京时间）。

修改触发时间 - 编辑 `config.json`：

```json
{
  "triggers": [{
    "config": "0 0 11 * * * *"
  }]
}
```

### 数据库集合

**集合名称**：`subscriptions`

**字段说明**：
```json
{
  "openId": "用户唯一标识",
  "subscribed": true,
  "city": "城市名称",
  "pushTime": "推送时间"
}
```

## 部署

1. 在微信开发者工具中右键 `pushWeather` 文件夹
2. 选择「上传并部署：云端安装依赖」
3. 等待部署完成

## 测试

在云开发控制台的云函数调试中运行，或通过定时触发器自动执行。
